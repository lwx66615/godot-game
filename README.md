## 如何运行

1. 使用 Godot 4.2+ 打开项目
2. 打开 `scenes/main.tscn`
3. 按 F5 运行

## 操作说明

| 按键 | 功能 |
|------|------|
| WASD | 移动 |
| Shift | 奔跑 |
| E | 交互（对话/开门/躲藏/拾取/搜刮） |
| Tab | 打开/关闭物品栏 |
| 方向键 | 物品栏导航 |
| Enter | 使用物品 |
| T | 跳过1小时（测试用） |

## 核心脚本

| 脚本 | 说明 |
|------|------|
| `scripts/player.gd` | 玩家控制、体力系统、躲藏状态 |
| `scripts/ghost.gd` | 诡异AI、状态机、感知系统、逃脱道具响应 |
| `scripts/game_manager.gd` | 时间系统、天数管理 |
| `scripts/game_ui.gd` | UI系统、交互提示、物品栏UI、对话系统 |
| `scripts/inventory.gd` | 物品栏管理、物品数据库、物品使用效果 |
| `scripts/item_pickup.gd` | 可拾取物品类 |
| `scripts/searchable.gd` | 可搜刮物体类 |
| `scripts/door.gd` | 门系统、敲门触发 |
| `scripts/hide_spot.gd` | 躲藏点系统 |
| `scripts/npc.gd` | NPC对话系统 |
| `scripts/audio_manager.gd` | 音频管理器 |

## 音效资源

当前使用程序化生成的占位符音效，可正常播放。

正式版本需要替换为专业音效资源，详见 `docs/音效资源清单.md`。

**音效目录结构：**
```
assets/audio/
├── ghost/      # 诡异音效（拖刀声、敲门声、破门声）
├── player/     # 玩家音效（脚步声、心跳声、呼吸声）
├── environment/ # 环境音效（门声、风声）
└── ui/         # UI音效
```
