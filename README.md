# 诡异末世 - 技术验证

## 项目状态

当前处于**技术验证阶段**，验证核心玩法机制。

## 验证内容

| 内容 | 状态 | 说明 |
|------|------|------|
| 玩家移动系统 | ✅ 完成 | 行走/奔跑/体力消耗 |
| 诡异状态机 | ✅ 完成 | 6种状态转换 |
| 感知系统 | ✅ 完成 | 视野/声音/距离衰减 |
| 敲门系统 | ✅ 完成 | 三阶段敲门 |

## 如何运行

1. 使用 Godot 4.2+ 打开项目
2. 打开 `scenes/main.tscn`
3. 按 F5 运行

## 操作说明

| 按键 | 功能 |
|------|------|
| WASD | 移动 |
| Shift | 奔跑 |
| E | 交互 |
| I | 物品栏 |
| ESC | 暂停 |

## 核心脚本

- `scripts/player.gd` - 玩家控制、体力系统
- `scripts/ghost.gd` - 诡异AI、状态机、感知系统
- `scripts/test_scene.gd` - 测试场景
