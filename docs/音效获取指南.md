## 二、获取方式

### 方式一：免费音效网站（推荐新手）

#### 1. Freesound.org（最推荐）

**优点**：
- 完全免费
- 音效种类丰富
- 有搜索和筛选功能

**步骤**：
1. 打开 https://freesound.org
2. 注册账号（免费）
3. 在搜索框输入关键词，如 "knife drag" 或 "door knock"
4. 使用筛选器：
   - Duration: Short sounds (< 1 sec) 或 Medium (1-10 sec)
   - License: Creative Commons 0（可商用）
5. 试听并下载 WAV 格式
6. **重要**：记录音效作者和许可证信息

**搜索关键词建议**：
| 中文 | 英文关键词 |
|------|------------|
| 拖刀声 | knife drag, metal scrape, blade drag |
| 敲门声 | door knock, knocking wood |
| 破门声 | door break, wood crash, door smash |
| 脚步声 | footsteps, walking indoor, footsteps wood |
| 心跳声 | heartbeat, heart beat |
| 开门声 | door open, door creak |
| 关门声 | door close, door slam |
| 风声 | wind, wind blowing, night wind |

#### 2. Pixabay（无需注册）

**优点**：
- 无需注册
- 可商用
- 质量较高

**步骤**：
1. 打开 https://pixabay.com/music/sound-effects/
2. 搜索关键词
3. 直接下载

#### 3. Mixkit（完全免费）

**优点**：
- 无需注册
- 可商用
- 有音效包

**步骤**：
1. 打开 https://mixkit.co/free-sound-effects/
2. 浏览分类或搜索
3. 点击下载

---

### 方式二：YouTube 音效频道

**推荐频道**：
- "Free Sound Effects" - 大量免费音效
- "Sound Effects Factory" - 各种音效合集

**步骤**：
1. 搜索 "door knock sound effect free"
2. 使用在线工具提取音频（如 y2mate.com）
3. 转换为 WAV 格式

**注意**：确保视频描述中标注了 "royalty free" 或 "free to use"

---

### 方式三：自制音效（进阶）

**工具**：
- Audacity（免费）：https://www.audacityteam.org/
- 录音设备：手机即可

**可以自制的音效**：
| 音效 | 方法 |
|------|------|
| 脚步声 | 在木地板上走动录音 |
| 开关门 | 录制真实的门开关声 |
| 敲门 | 敲击木门或桌子 |
| 心跳 | 用手敲击胸口模拟 |

**后期处理步骤**：
1. 用 Audacity 打开录音
2. 剪辑到合适长度
3. 降噪（效果 → 降噪）
4. 调整音量（效果 → 放大）
5. 导出为 WAV（16-bit PCM）

## 三、下载后的处理

### 1. 文件格式转换

**推荐格式**：
- 短音效（< 1秒）：WAV（无损，加载快）
- 循环音效：OGG（压缩好，循环支持）

**转换工具**：
- 在线：https://convertio.co/zh/wav-ogg/
- 本地：Audacity

### 2. 文件命名

按照 AudioManager 中的名称命名：
```
assets/audio/ghost/drag_knife.wav
assets/audio/ghost/knock_slow.wav
assets/audio/ghost/knock_medium.wav
assets/audio/ghost/knock_fast.wav
assets/audio/ghost/breach.wav
assets/audio/player/footstep_walk.wav
assets/audio/player/footstep_run.wav
assets/audio/player/heartbeat_slow.wav
assets/audio/player/heartbeat_fast.wav
assets/audio/environment/door_open.wav
assets/audio/environment/door_close.wav
assets/audio/environment/wind_night.ogg
assets/audio/ui/ui_prompt.wav
assets/audio/ui/ui_warning.wav
```

### 3. 放入项目

将下载的文件放入对应目录：
```
godot-game/
└── assets/
    └── audio/
        ├── ghost/      ← 诡异音效放这里
        ├── player/     ← 玩家音效放这里
        ├── environment/ ← 环境音效放这里
        └── ui/         ← UI音效放这里
```

**放入后自动生效**：AudioManager 会自动检测并使用这些文件。

## 五、快速开始（5分钟教程）

### 最简单的开始方式

1. **打开** https://freesound.org
2. **注册**一个免费账号
3. **搜索** "door knock"
4. **筛选** Duration < 1 second
5. **试听**几个，选择喜欢的
6. **下载** WAV 格式
7. **重命名**为 `knock_slow.wav`
8. **放入** `assets/audio/ghost/` 目录
9. **运行游戏**测试

AudioManager 会自动使用新文件，无需修改代码！

## 六、常见问题

### Q: 下载的音效太大声/太小声怎么办？
A: 用 Audacity 调整音量，或在 AudioManager 的 sound_configs 中调整 volume_db 值。

### Q: 找不到合适的音效怎么办？
A: 尝试不同的搜索关键词，或组合多个音效。

### Q: 循环音效有爆音怎么办？
A: 在 Audacity 中添加淡入淡出（效果 → 淡入/淡出）。

### Q: 可以用 AI 生成音效吗？
A: 可以尝试：
- ElevenLabs Sound Effects（付费）
- AudioCraft by Meta（免费，需本地运行）
