# 音效资源清单

本文档列出了游戏所需的所有音效资源，包括占位符状态和正式资源规格。

---

## 音效状态说明

| 状态 | 说明 |
|------|------|
| 🟡 占位符 | 已有程序化生成的占位符音效，需要替换为正式资源 |
| 🟢 正式 | 已有正式音效资源 |
| 🔴 缺失 | 尚未实现 |

---

## 一、诡异音效 (ghost/)

| 音效名称 | 状态 | 描述 | 规格要求 |
|----------|------|------|----------|
| drag_knife | 🟡 占位符 | 拖刀声 | 循环、距离衰减、金属摩擦质感 |
| knock_slow | 🟡 占位符 | 敲门声(慢) | 低沉撞击、2秒间隔 |
| knock_medium | 🟡 占位符 | 敲门声(中) | 中等撞击、1秒间隔 |
| knock_fast | 🟡 占位符 | 敲门声(快) | 急促撞击、0.5秒间隔 |
| breach | 🟡 占位符 | 破门声 | 剧烈撞击、木头碎裂 |
| search_pace | 🟡 占位符 | 搜索徘徊声 | 循环、轻微脚步/摩擦声 |

### 正式资源建议

**拖刀声 (drag_knife)**
- 时长：0.5-1秒循环
- 特点：金属拖地、有节奏感
- 参考：恐怖游戏中的金属摩擦声

**敲门声 (knock_*)**
- 三阶段递进：慢→中→快
- 材质：木门撞击
- 氛围：不祥、压迫感

**破门声 (breach)**
- 时长：0.3-0.5秒
- 特点：剧烈、突然
- 元素：木头碎裂、撞击

## 二、玩家音效 (player/)

| 音效名称 | 状态 | 描述 | 规格要求 |
|----------|------|------|----------|
| footstep_walk | 🟡 占位符 | 脚步声(走) | 轻微、室内地面 |
| footstep_run | 🟡 占位符 | 脚步声(跑) | 较重、急促 |
| heartbeat_slow | 🟡 占位符 | 心跳声(慢) | 循环、紧张但不急促 |
| heartbeat_fast | 🟡 占位符 | 心跳声(快) | 循环、急促紧张 |
| breath_normal | 🟡 占位符 | 呼吸声(正常) | 循环、轻微 |
| breath_stress | 🟡 占位符 | 呼吸声(紧张) | 循环、急促 |

### 正式资源建议

**脚步声**
- 材质：室内地板/瓷砖
- 变体：建议2-3种随机变体
- 时长：0.08-0.1秒

**心跳声**
- 慢速：约60-70 BPM
- 快速：约100-120 BPM
- 特点：低沉、有回响

**呼吸声**
- 正常：平静、规律
- 紧张：急促、略带颤抖

| 音效名称 | 状态 | 描述 | 规格要求 |
|----------|------|------|----------|
| door_open | 🟡 占位符 | 开门声 | 木门、有吱呀感 |
| door_close | 🟡 占位符 | 关门声 | 木门、有撞击感 |
| door_creak | 🟡 占位符 | 门吱呀声 | 诡异、自动开关 |
| wind_day | 🟡 占位符 | 风声(白天) | 循环、轻微、孤寂 |
| wind_night | 🟡 占位符 | 风声(夜晚) | 循环、凄厉、压抑 |
| light_flicker | 🟡 占位符 | 灯光闪烁 | 电流声、短促 |

### 正式资源建议

**门音效**
- 材质：老式木门
- 特点：有重量感、略带生锈铰链
- 变体：建议2种变体

**风声**
- 白天：轻微、有鸟鸣暗示
- 夜晚：凄厉、有诡异感
- 时长：1-2秒循环

**灯光闪烁**
- 时长：0.05-0.1秒
- 特点：电流滋滋声

## 六、音效资源获取建议

### 免费资源网站

1. **Freesound.org** - 大量免费音效
2. **Zapsplat** - 需注册，免费版有限制
3. **Mixkit** - 完全免费

### 付费资源

1. **Unity Asset Store** - 音效包
2. **Unreal Marketplace** - 音效包
3. **专业音效库** - 如 Boom Library

### 自制建议

- 使用 Audacity 进行音效编辑
- 录制真实声音作为基础
- 使用合成器生成氛围音效

## 七、音效文件格式建议

| 格式 | 优点 | 缺点 | 推荐场景 |
|------|------|------|----------|
| WAV | 无损、加载快 | 文件大 | 短音效 |
| OGG | 压缩好、循环支持 | 有损 | 背景音乐、长音效 |
| MP3 | 兼容性好 | 压缩质量一般 | 不推荐 |

**推荐**：短音效用 WAV，循环音效用 OGG
